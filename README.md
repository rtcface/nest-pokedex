# Run in development
1. Clone the repository
2. Run
```
yarn install
```
3. Have nestjs cli intalled.
```
npm i -g @nest/cli 
```
4. Start the database
```
docker-compose up -d
```
5. rebuild database with see
```
localhost:3000/api/v2/seed
```

## Used Stack 
* Mongo DB
* Nest
